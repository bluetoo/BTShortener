BTShortener is a simple URL shortening library for iOS. Its goal is to make adding URL shortening to apps quick and easy.

Patches are very welcomed...

Requirements
------------
1. [asi-http-request][asi]
2. [json-framework][jsonframework]
3. iOS 4+ (blocks)
